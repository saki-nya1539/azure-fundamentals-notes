# azure-fundamentals-notes

Microsoft Azureの基礎知識をまとめたノートです。AWSでの学習・ハンズオン経験(`aws-fundamentals-notes`)をベースに、AWSとAzureの概念対応、責任共有モデル、Microsoft Entra ID(旧Azure AD)とテナントの権限構造などを整理しています。

## 内容

- AWSとAzureの主要サービス対応表(VM/EC2、NSG/セキュリティグループ、Entra ID/IAM など)
- Azure for Students(学生向け無料枠)の仕組みと制約
- 責任共有モデル・課金の考え方
- Entra IDとテナント・サブスクリプションの権限分離構造
- 実際のハンズオン作業との対応関係
